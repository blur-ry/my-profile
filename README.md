# Portfolio Website

Modern portfolio website built with Next.js, TypeScript, React, and Supabase.

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Open `http://localhost:3000`

## Architecture

- `src/app`: App Router pages and API routes
- `src/components`: UI sections and shared components
- `src/lib`: Supabase client and SEO utilities
- `src/services`: API helpers
- `src/types`: shared TypeScript models
- `src/utils`: small helper functions
- `supabase/schema.sql`: table definitions for Supabase
